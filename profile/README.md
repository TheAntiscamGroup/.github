The AntiScam Group is a joint operation created by @[SocksTheWolf](https://github.com/socksthewolf) to deal with some of the rampant scams that have been plaguing our communities. It is comprised of server admins, moderators and users reporting the scams they receive over Discord.

TAG maintains and operates the Discord bot, [ScamGuard](https://scamguard.app) for the community.
